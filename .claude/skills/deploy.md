# Skill: deploy

Verify, build, and deploy the site to the configured hosting platform.

## Usage
```
/deploy staging
/deploy production
```

## Pre-deploy Checklist
1. All tests pass (`npm test`)
2. Build succeeds locally (`npm run build`)
3. No secrets in source code (`git grep -r "sk_" src/` etc.)
4. Environment variables are set in hosting platform
5. `docs/architecture.md` is up to date

## Staging Deploy
1. Push to `dev` branch
2. CI runs automatically
3. Preview URL generated by hosting platform
4. Verify site manually on preview URL

## Production Deploy
1. Confirm all staging checks passed
2. Merge `dev` → `main` via PR
3. CI deploys to production automatically
4. Verify production URL
5. Check Lighthouse score post-deploy
